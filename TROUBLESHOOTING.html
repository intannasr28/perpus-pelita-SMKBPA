⚠️ Common Issues & Solutions

Issue 1: Avatar Masih Muncul Perempuan Padahal Laki-laki

Penyebab Kemungkinan:
  • File avatar_laki_laki.png adalah duplikat (isinya sama dengan perempuan)
  • Data jekel di database kosong atau format tidak standard
  • Cache browser
Solusi:
1️⃣ Buka check-avatars.php
2️⃣ Jika "DUPLIKAT TERDETEKSI" → Ganti file avatar_laki_laki.png
    - Download avatar laki-laki dari dicebear.com
    - Rename: avatar_laki_laki.png
    - Upload ke: dist/img/
3️⃣ Buka debug-avatar.php
4️⃣ Jika jekel kosong → Edit profile siswa, set jenis kelamin
5️⃣ Clear cache: Ctrl+Shift+Delete → All time → Clear
6️⃣ Refresh: Ctrl+F5

Issue 2: Level Pengguna Tidak Terlihat

Penyebab: HTML structure yang broken (sudah di-fix) Solusi:
1️⃣ Full refresh browser: Ctrl+Shift+Delete (clear cache)
2️⃣ Close browser sepenuhnya
3️⃣ Buka ulang dan login kembali
4️⃣ Check sidebar - sekarang level harus terlihat ✓

🧪 Testing Manual

Test 1: Check Sidebar Avatar & Level
  1. Login sebagai user PEREMPUAN
  2. Lihat sidebar sebelah kiri
  3. Verify:
      ✓ Avatar terlihat (gambar perempuan)
      ✓ Nama user terlihat
      ✓ Level terlihat (misal: Siswa, Petugas, Administrator)
Test 2: Check Male User
  1. Logout (atau buka private window)
  2. Login sebagai user LAKI-LAKI
  3. Verify:
      ✓ Avatar BERBEDA dari perempuan (harus laki-laki)
      ✓ Nama user terlihat
      ✓ Level terlihat
Test 3: Check Profile Page
  1. Selama login, klik nama user → Profile Siswa
  2. Verify:
      ✓ Avatar di "Foto Profil" sesuai jenis kelamin
      ✓ Jenis kelamin tertulis di bawah avatar
      ✓ Data profil lengkap (ID, Nama, Jenis Kelamin, Kelas, HP)
      ✓ Level terlihat di "Informasi Akun"

🎨 Avatar Resources

Recommended Avatar Generator:
🔗 Dicebear.com

How to use:
  1. Go to https://www.dicebear.com/
  2. Select style "Avataaars"
  3. Untuk laki-laki: Select gender filter "male"
  4. Untuk perempuan: Select gender filter "female"
  5. Download PNG
  6. Rename: avatar_laki_laki.png atau avatar_perempuan.png
  7. Upload ke: dist/img/
